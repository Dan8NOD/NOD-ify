# NOD-ify — MixMatch FV4

Negotiation tools deck app with three interactive views: Tool Deck, Change Stair, and Emotion Wheel.

## Views

### 🃏 Tool Deck
23+ negotiation tactics across 5 categories (Rapport, Information, Emotion, Strategy, Closure). Start a timed session and draw tools at random. Cards spread on a green felt table with gold accents.

### 🏔 Change Stair
The Machiavelli Behaviour Change model — 4 stages with Rapport flowing across all:
- **GET INFORMATION** → **EMPATHY** → **TRUST** → **COLLABORATION**
- Click any stage card to expand full description and tactic list
- Staggered reveal animation

### 🧠 Emotion Wheel
Custom 6-emotion concentric wheel (ANGER, DISGUST, SAD, HAPPY, SURPRISE, FEAR):
- 3 rings: primary center → secondary middle → tertiary outer
- Tap any sector for full breakdown with intensity badges
- Drag or slider to rotate
- Gold pointer and rim dots with casino-table styling

## Features
- Timed sessions with color-coded countdown (warning → danger)
- **Stop** button — end any session cleanly with full summary
- **Skip** button — skip cards without breaking the session
- End-of-session summary: tools drawn, skipped, unique frames, total time, last emotion
- Session log with skipped-item tracking (strikethrough)
- Category filter pills with smooth horizontal scroll
- Add custom tools on the fly
- Card detail popovers
- Mobile responsive: 2-column card grid, tappable buttons (min 40px), horizontal slider for wheel
- Dark felt table UI with gold accent (#d4a853)
